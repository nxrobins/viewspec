# Launch Headliner Asset

The launch headliner asset shows the proof chain: prompt -> one IntentBundle -> HTML/Tailwind, React TSX, SwiftUI, and Flutter outputs.

## Files

- MP4: `demos/launch-assets/headliner-prompt-four-outputs.mp4`
- GIF fallback: `demos/launch-assets/headliner-prompt-four-outputs.gif`
- Poster: `demos/launch-assets/headliner-prompt-four-outputs.png`
- Manifest: `demos/launch-assets/headliner-manifest.json`

## Source Inputs

- Prompt: `demos/cross-platform-dashboard/artifacts/agent_prompt.txt`
- IntentBundle: `demos/cross-platform-dashboard/artifacts/intent_bundle.json`
- ASTBundle: `demos/cross-platform-dashboard/artifacts/ast_bundle.json`
- Generated sources: HTML/Tailwind, React TSX, SwiftUI, Flutter.

## Usage Guidance

- Use the MP4 in launch pages or social previews when video is supported.
- Use the GIF fallback for README, email, and surfaces that do not autoplay MP4.
- Do not embed SwiftUI/Flutter runtime recordings until external simulator/emulator videos are delivered.
